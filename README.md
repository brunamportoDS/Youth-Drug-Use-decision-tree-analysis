# Youth-Drug-Use-decision-tree-analysis

Analysis of youth substance use using decision tree and ensemble methods 
on the NSDUH 2023 survey data.

## Research Question
What youth experiences and demographic factors predict substance use among minors?

## Methods
- Binary classification (marijuana use)
- Multi-class classification (alcohol frequency)
- Regression (alcohol use days)
- Models: Decision Tree, Bagging, Random Forest, Gradient Boosting

## Data
Data from the National Survey on Drug Use and Health (NSDUH 2023)
